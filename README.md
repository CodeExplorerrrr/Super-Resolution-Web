### 1. Clone the Repository  
Copy the following command to clone the project:
```bash

```

### 2. Install Dependencies  
Navigate to the project directory and install required dependencies:  
```bash
cd SR-Inference-Web
npm install
```

### 3. Run the Project  
Start the development server:  
